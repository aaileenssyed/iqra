# IQRA Arabic Academy LMS Portal

A simple lightweight LMS-style portal for IQRA Arabic Academy.

This project uses:

- **GitHub Pages** for hosting the frontend
- **Google Sheets** as the database
- **Google Apps Script** as the backend API
- **HTML, CSS, and JavaScript** for the website

The portal is intentionally simple. It does not include a syllabus module, lesson tracking, user accounts, or complex LMS features. It focuses only on student reports, exams, subjects, marks, parameters, and sub-parameters.

---

## Features

### Student Report Portal

Students can:

- Search using Student ID or name
- Select their enrolled class
- Select a subject
- Select an exam
- View marks by parameter and sub-parameter
- View total marks and percentage

### Teacher Portal

Teachers can:

- Add classes
- Add subjects under classes
- Add students
- Enroll students into classes
- Add exams for subjects
- Add marking parameters
- Add sub-parameters inside parameters
- Enter marks for each student
- View a quick class marksheet
